# 🔑 支持系统

软件全系支持 win7,win8,win10、win11 系统。（ **推荐使用 Win10及以上专业版系统** ）

QIKI系列软件额外支持 MAC 系统，每一款软件都有3个软件包，win32位，win64 位和 MAC。
下载软件时请下载对应的那个软件包。
